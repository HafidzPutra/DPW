About Me

Perkenalkan nama saya Hafidz Putra Zainal. Saya adalah mahasiswa Program Studi S1 Teknik Informatika, Fakultas Teknik, Universitas Riau. Saya lahir di kota Pekanbaru pada tanggal 23 Januari 2003. Saya mempunyai satu kakak dan satu abang. Saya merupakan anak ketiga dari tiga bersaudara . Saya mempunyai beberapa hoby yaitu seperti menonton film, membaca buku cerita, dan ensiklopedia. Ketika ada waktu luang saya biasanya membaca buku atau tidur. Hal ini saya lakukan untuk menghilangkan rasa bosan. Awal mula saya menempuh pendidikan yaitu di SDN 112 Pekanbaru. Saya menjalani pendidikan disana selama 6 tahun. Kemudian saya melanjutkan pendidikan ke SMPN 34 Pekanbaru selama 3 tahun. Lalu saya melanjutkan pendidikan di SMAN 4 Pekanbaru. Setelah lulus SMA saya mengikuti SBMPTN agar dapat melanjutkan pendidikan di perguruan tinggi. Alhamdullillah saya lulus masuk Program Studi S1 Teknik Informatika, Fakultas Teknik, Universitas Riau. Saya ingin menjadi seorang programmer. Saya ingin membuat berbagai aplikasi yang dapat digunakan untuk membantu berbagai kegiatan manusia. selain itu aplikasi ini nantinya juga akan saya rilis di berbagai OS sehingga saya juga mendapatkan keuntungan. Oleh karena itu, selama menempuh pendidikan perguruan tinggi ini, saya akan belajar dengan sungguh-sungguh dan berusaha mempelajari segala hal yang nantinya akan saya perlukan ketika sudah masuk ke dunia kerja.

Hal yang diharapkan setelah lulus :

  1. Bekerja di perusahaan besar ternama dan mendapatkan gaji yang besar
  2. Mampu membuat aplikasi yang dapat digunakan banyak orang
  3. Mengusai berbagai macam bahasa pemrograman
